Sobre o PUC-Go

O PUC-Go é uma plataforma interativa desenvolvida para auxiliar alunos, funcionários e visitantes a se localizarem no campus da PUC-Rio, tornando a experiência acadêmica mais prática e eficaz.

Nossa Missão

Facilitar a navegação pelo campus universitário através de uma interface intuitiva e recursos modernos de mapeamento, proporcionando economia de tempo e melhor aproveitamento das instalações disponíveis.

Por que o PUC-Go foi criado?

O campus da PUC-Rio é extenso e possui diversos prédios, laboratórios, salas e espaços. Para novos alunos e visitantes, encontrar um local específico pode ser desafiador.

O PUC-Go nasceu como um projeto da disciplina INF-1039, com o objetivo de resolver esse problema de forma tecnológica e acessível.

Funcionalidades Principais

  • Pesquisa de Locais: Encontre rapidamente salas de aula, laboratórios, restaurantes, banheiros e outros espaços
  • Mapa Interativo: Visualize o campus com marcadores precisos de cada prédio
  • Rotas: Trace caminhos do seu ponto atual até o destino desejado
  • Favoritos: Salve seus locais mais visitados para acesso rápido
  • Filtros Avançados: Busque por categoria, prédio e características de acessibilidade

Tecnologias Utilizadas

O PUC-Go foi desenvolvido utilizando tecnologias modernas e confiáveis:

  • Django: Framework web Python para o backend
  • Leaflet.js: Biblioteca JavaScript para mapas interativos
  • OpenStreetMap: Dados cartográficos de código aberto
  • SQLite: Banco de dados para armazenamento de informações
